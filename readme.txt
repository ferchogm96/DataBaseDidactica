Necesitamos instalar XAMPP, que es el responsable de descargar e instalar php, mysql y apache
También necesitaremos npm ya que asi correremos nuestro proyecto usando el comando en terminal "npm run dev"
Una vez descargado xampp, se guardará en Disco Local C
Y nuestro proyecto estará ubicado en Disco Local C/xampp/htdocs/DataBaseDidactica
